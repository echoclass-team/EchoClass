# 30 秒电梯版

走廊 / 电梯里被问"你做的什么"时用。6 句话，90 字内，一口气说完。学术口径，不出现 TAM / GTM。

## 背稿

> EchoClass —— 面向师范生的 1v1 答疑陪练系统。
>
> 师范生在微格教学里对着同班同学试讲，提不出真儿童式困惑，这是国内教师培养里长期被忽视的环节。
>
> 我们把皮亚杰认知阶段约束写进 prompt，把 21 份学科迷思库嵌进 RAG，让大模型扮演一个会犯典型错误、不被讲到点上不说"懂了"的虚拟学生。
>
> 上传教案就能开练，结束后按 5 维 Rubric 出诊断报告 + 自然语言反馈。
>
> 华师大跑了 _<n>_ 名师范生测试，SUS _<xx>_ 分，NPS _<x>_。
>
> 第 _<x>_ 场答辩，demo 区见。

## 打断后只说两句

> 师范生答疑陪练 —— 让大模型扮演会犯典型错误、不被讲到点上不说"懂了"的虚拟学生，让师范生没有真学生也能练答疑。

## 追问速查

| 问题 | 一句答 |
|---|---|
| 为什么不用 ChatGPT | 通用 LLM 让它扮小学生会"超模"。我们用学段约束 + 21 份迷思库注入儿童式错误前提 |
| 和市面上 AI 教育差别 | 市面几乎全是 To-Student 辅导。我们做 To-Teacher 训练，是空白赛道 |
| 理论支撑 | 皮亚杰认知阶段、维果茨基 ZPD + 脚手架，都落到 prompt 与数据层 |
| 技术亮点 | 单 Agent 二阶段（宽生成 + self-check）+ 流式 holdback 缓冲剥离 `[懂了]` |
| 验证证据 | _<n>_ 人闭环测试，金句：'_<待回填>_' |
| 下一步 | Rubric v0.1 迭代 + 师范学院试点 + 教研员沉淀语料 |
